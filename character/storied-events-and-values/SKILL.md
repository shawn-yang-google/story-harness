# Storied Events and Values

Design storied events as meaningful moments of value-charged change.

- Identify human values as binaries of experience that can change charge (e.g., Life/Death, Justice/Injustice, Love/Hate).
- Create meaning by imbuing every story event with a value shift.
- Ensure a storied event reverses the charge of a value at stake (from positive to negative or vice versa).
- Recognize that an event is meaningless if it changes nothing of value to the characters.
- Use character actions or reactions to trigger these value reversals.
- Examples of value-charged changes include:
    - Love (+) to Hate (-)
    - Poverty (-) to Wealth (+)
    - Freedom (+) to Slavery (-)
- Anchor the audience to the plot through the meaningful changes experienced by the characters.
