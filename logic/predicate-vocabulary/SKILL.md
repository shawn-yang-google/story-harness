# Predicate Vocabulary

## Vocabulary Items

| Category | Symbols | Notes |
|----------|---------|-------|
| **Names** | a, b, c, d, a', b', c', d', a₂, ... | Refer to objects |
| **Variables** | u, v, w, x, y, z, u', v', ... | Used by quantifiers |
| **1-place predicates** | A¹, ..., Z¹ | Properties (superscripts usually omitted) |
| **2-place predicates** | A², ..., Z² | Binary relations |
| **n-place predicates** | Aⁿ, ..., Zⁿ | n-ary relations |
| **Identity symbol** | `=` | Special 2-place predicate, placed between names |
| **Universal quantifier** | `∀α` | "every" (α is a variable) |
| **Existential quantifier** | `∃α` | "some" (α is a variable) |

Plus sentence letters, connectives, and parentheses from sentential logic.

## Disambiguation

A capital letter with:
- No names/variables → sentence letter (e.g., `P`)
- 1 name/variable → 1-place predicate (e.g., `Fa`)
- 2 names/variables → 2-place predicate (e.g., `Rab`)
- n names/variables → n-place predicate

## Metavariables

Greek letters α, β, γ are metavariables for names and variables of predicate logic.
