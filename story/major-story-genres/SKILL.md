# Major Story Genres

[MASTER_THEORIST]'s genre taxonomy — each genre is defined by its **core values** and **conventions**.

## The Genres
| Genre | Core Value |
|-------|-----------|
| **Love Story** | Love/hate, intimacy/isolation |
| **Horror** | Life/death, survival |
| **Modern Epic** | Individual vs state, freedom/tyranny |
| **Western** | Civilization/wilderness, law/anarchy |
| **War** | Life/death, patriotism/disillusionment |
| **Maturation Plot** | Growth/stagnation, innocence/experience |
| **Redemption Plot** | Good/evil within the self |
| **Punitive Plot** | Justice/injustice, guilt/innocence |
| **Testing Plot** | Courage/cowardice, will/surrender |
| **Education Plot** | Knowledge/ignorance, meaning/meaninglessness |
| **Disillusionment Plot** | Idealism/cynicism, hope/despair |
| **Comedy** | Various — defined by tone and resolution |
| **Crime** | Justice/injustice, legality/criminality |
| **Social Drama** | Individual vs institution |
| **Action/Adventure** | Life/death, excitement/boredom |

## Mega-Genres and Sub-Genres
Genres contain **sub-genres** (e.g., Crime → Detective, Gangster, Thriller, Courtroom, Newspaper, etc.) and can be combined into **mega-genres** that blend conventions from multiple genres.

## Key Principle
Genre is not a limitation but a **creative challenge**. Be honest in your choice of genre — of all reasons for wanting to write, only **love of the work itself** nurtures us through time.
